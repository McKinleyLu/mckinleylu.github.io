快速排序

快速排序是在一般情况下时间复杂度为O(nlogn)的排序。

基本思想

通过一趟排序将待排记录分隔成独立的两部分，其中一部分记录的关键字均比另一部分的关键字小，则可分别对这两部分记录继续进行排序，以达到整个序列有序。

时间复杂度

  1. 当情况为最优时(即基准每一次都在数组中央)，则

    比较次数（每一次都比较n次）为：

    T(n) <= 2*T(n/2)+n

    T(n) <= 2(2(T(n/4) + n / 2) + n = 4T(n/4) + 2n

    ……

    T(n) <= nT(1) + nlog2n = O(nlogn)

  2. 但情况最糟糕时

    比较次数 = (n - 1) + (n - 2 ) + … + 2 + 1 = n(n-1) / 2 ;

    O(n^2)
